
# ANC Global Trading — B2B Corporate Website

## Design System
- **Dark theme**: Deep navy (#0A1628) primary background, near-black (#060D1A) for alternating sections
- **Accent**: Subtle blue highlights (#1E3A5F) matching the brand's navy eagle logo
- **Typography**: Clean sans-serif (Inter), strong size hierarchy, white text on dark
- **Spacing**: Generous padding, minimal clutter, premium feel
- **Animations**: Subtle fade-in on scroll, smooth parallax on hero image
